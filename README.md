## Pembagian IP
  Pemberian label `AX` pada topologi
  
  Mengatur IP pada setiap interface berdasarkan subnet yang telah ditentukan
  - Router
  - Client
  - Server
  
## Routing
  Melakukan routing pada setiap interface router menuju subnet yang belum dikenal
